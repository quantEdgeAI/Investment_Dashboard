# Quick User Guide - Investment Analytics Dashboard

## Table Data Viewer - Sorting Feature

### How to Sort Columns

**Click any column header to sort the data:**

1. **First Click**: Sorts ascending (A to Z, 0 to 9, oldest to newest)
2. **Second Click**: Sorts descending (Z to A, 9 to 0, newest to oldest)
3. **Third Click**: Toggles back to ascending

### Visual Indicators

- **⇅ (up-down arrows)**: Column is not sorted (shows on hover)
- **↑ (up arrow)**: Sorted in ascending order
- **↓ (down arrow)**: Sorted in descending order

### Examples

**Sort by Signal ID:**
- Click "signalId" → Shows 1, 2, 3, 4... (ascending)
- Click again → Shows 4, 3, 2, 1... (descending)

**Sort by Symbol:**
- Click "symbol" → Shows ANGELONE, MUTHOOT... (A to Z)
- Click again → Shows MUTHOOT, ANGELONE... (Z to A)

**Sort by Date:**
- Click "date" → Shows oldest dates first
- Click again → Shows newest dates first

### Tips

- Hover over column headers to see sort icons
- Sorting works with filters and pagination
- Sort state persists when navigating pages
- Changing tables resets sorting to default
